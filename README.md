# MSA - Spring Boot

## URI
1. /api/authorize => 로그인
2. /api/authorizeTokenRefresh => 토큰 갱신 ( 로그인 후 만기된 accessToken을 refreshToken으로 갱신)
3. /api/register => 간단한 회원가입 ( 테스트 용 )
4. /api/selectAllUser => Material React Table 테스트를 위한 모든정보

## VERSION
1. Spring Boot 3.0.6 => REST API
2. Java EE 17
3. Maria DB
